# Newsletter Email Templates Reference

This document describes the HTML email template structure used for Thailand Hayom newsletters.

## Template Structure

The newsletter HTML template is generated by `server/newsletterRouter.ts` in the `generateNewsletterHTML()` function.

### Layout Components

```
┌─────────────────────────────────┐
│ Header (Blue Gradient)          │
│ - Logo/Title                     │
│ - Tagline (bilingual)            │
└─────────────────────────────────┘
┌─────────────────────────────────┐
│ Custom Content (Yellow Banner)   │
│ - Hebrew version (RTL)           │
│ - Optional                       │
└─────────────────────────────────┘
┌─────────────────────────────────┐
│ Articles Section                 │
│ ┌─────────────────────────────┐ │
│ │ Article Card                 │ │
│ │ - Premium badge (optional)   │ │
│ │ - Title (Hebrew)             │ │
│ │ - Title (English)            │ │
│ │ - Excerpt (Hebrew)           │ │
│ │ - Read More button           │ │
│ └─────────────────────────────┘ │
│ [More article cards...]          │
└─────────────────────────────────┘
┌─────────────────────────────────┐
│ Events Section                   │
│ ┌─────────────────────────────┐ │
│ │ Event Card                   │ │
│ │ - Premium badge (optional)   │ │
│ │ - Title (Hebrew)             │ │
│ │ - Title (English)            │ │
│ │ - Meta (date, location, $)   │ │
│ │ - Description excerpt        │ │
│ │ - Registration button        │ │
│ └─────────────────────────────┘ │
│ [More event cards...]            │
└─────────────────────────────────┘
┌─────────────────────────────────┐
│ Custom Content (Yellow Banner)   │
│ - English version (LTR)          │
│ - Optional                       │
└─────────────────────────────────┘
┌─────────────────────────────────┐
│ Footer (Light Gray)              │
│ - Thank you message              │
│ - Links (Website, Articles, etc)│
│ - Unsubscribe link               │
└─────────────────────────────────┘
```

## Color Scheme

### Primary Colors
- **Header Background**: Linear gradient `#2563eb` (blue) to `#1e40af` (darker blue)
- **Section Titles**: `#2563eb` (blue)
- **Body Background**: `#f5f5f5` (light gray)
- **Card Background**: `#ffffff` (white)
- **Footer Background**: `#f9fafb` (very light gray)

### Accent Colors
- **Premium Badge**: `#f59e0b` (amber/gold)
- **Custom Content Banner**: `#fef3c7` (light yellow) with `#f59e0b` (amber) left border
- **Read More Button**: `#2563eb` (blue) background, white text
- **Link Hover**: `#2563eb` (blue)

### Text Colors
- **Primary Text**: `#1f2937` (dark gray)
- **Secondary Text**: `#6b7280` (medium gray)
- **Footer Text**: `#9ca3af` (light gray)

## Typography

### Fonts
Primary font stack:
```css
font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
```

### Font Sizes
- **Header Title**: 28px, bold
- **Header Tagline**: Inherited, opacity 0.9
- **Section Titles**: 22px, bold
- **Article/Event Titles**: 18px, bold
- **Body Text**: 16px (inherited)
- **Meta Text**: 14px
- **Footer Text**: 12px

## RTL Support

The template supports both RTL (Hebrew) and LTR (English) content:

### RTL Implementation
```html
<html dir="rtl">
  <!-- Hebrew content flows right-to-left -->
  <div dir="rtl">תוכן בעברית</div>

  <!-- English content flows left-to-right -->
  <div dir="ltr">English content</div>
</html>
```

### Best Practices
- Set `dir="rtl"` on root `<html>` tag for default RTL
- Override with `dir="ltr"` for English sections
- Use logical CSS properties when possible (e.g., `margin-inline-start` instead of `margin-left`)
- Test in both RTL and LTR email clients

## Responsive Design

The template uses max-width containers for mobile compatibility:

```css
.container {
  max-width: 600px;
  margin: 0 auto;
}
```

### Mobile Considerations
- Single column layout (no complex grids)
- Touch-friendly button sizes (min 44x44px)
- Readable font sizes (min 14px for body text)
- Sufficient padding for touch targets

## Email Client Compatibility

### Supported Clients
- ✅ Gmail (web, iOS, Android)
- ✅ Apple Mail (macOS, iOS)
- ✅ Outlook (2016+, web)
- ✅ Yahoo Mail
- ✅ ProtonMail

### Known Limitations
- ⚠️ Outlook 2007-2013 (limited CSS support)
- ⚠️ Older Android email clients (may not render gradients)

### Safe CSS Features
- Inline styles (always preferred)
- Basic colors (hex, named colors)
- Simple borders and padding
- Basic text styles (bold, italic, underline)

### Avoid
- ❌ External stylesheets
- ❌ JavaScript
- ❌ Video/audio embeds
- ❌ Complex animations
- ❌ CSS Grid or Flexbox (limited support)

## Dynamic Content

### Article Cards

Each article includes:
```typescript
{
  id: number
  titleHe: string  // Hebrew title
  title: string    // English title
  excerptHe?: string  // Hebrew excerpt (optional)
  excerpt?: string    // English excerpt (optional)
  slug: string     // URL slug
  isPremium: boolean
  category: string
  coverImage?: string  // Cover image URL (optional)
  publishedAt: Date
}
```

### Event Cards

Each event includes:
```typescript
{
  id: number
  titleHe: string  // Hebrew title
  title: string    // English title
  descriptionHe?: string  // Hebrew description (optional)
  description?: string    // English description (optional)
  eventDate: Date
  location?: string
  locationHe?: string  // Hebrew location (optional)
  price?: number   // In ILS (₪)
  isPremium: boolean
  registrationUrl?: string
}
```

## Customization

### Adding Custom Sections

To add new sections to the template:

1. Edit `server/newsletterRouter.ts`
2. Modify `generateNewsletterHTML()` function
3. Add new section HTML with consistent styling:

```typescript
${showNewSection ? `
  <h2 class="section-title">New Section Title</h2>
  <div class="custom-card">
    <!-- Custom content here -->
  </div>
` : ''}
```

### Styling Guidelines

When adding custom styles:
- Use inline CSS only
- Match existing color scheme
- Maintain responsive padding/margins
- Test in multiple email clients
- Ensure RTL compatibility
